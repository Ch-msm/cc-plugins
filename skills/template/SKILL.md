# 技能模板

这是一个用于创建新 Claude Code 技能的模板文件。

## 使用方法

1. 复制此模板到新的技能文件夹
2. 修改顶部的 YAML frontmatter：
   - `name`: 技能的唯一标识符（小写，用连字符分隔）
   - `description`: 技能的完整描述，说明何时使用
3. 更新技能说明和使用指南
